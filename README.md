## 📖 เกี่ยวกับโปรเจกต์ (Overview)

**AETHERIUM PLATFORM** ไม่ใช่เพียงแค่ Software Project แต่คือ **"ระบบจำลองสังคมดิจิทัล" (Digital Society Simulation)** ที่ขับเคลื่อนด้วย **Multi-Agent System** ภายใต้กฎระเบียบที่เข้มงวด (Governance Protocols)

หัวใจสำคัญของระบบคือการผสาน **Logic (ตรรกะ)** เข้ากับ **Ethics (จริยธรรม)** ผ่านระบบภูมิคุ้มกันดิจิทัล (Digital Immune System) ที่คอยตรวจสอบทุกธุรกรรมและการกระทำ เพื่อให้มั่นใจว่าเป็นไปตามปรัชญา "Collective Understanding"

## 🏗️ สถาปัตยกรรมระบบ (System Architecture)

ระบบทำงานบนโครงสร้าง **Event-Driven Architecture** โดยมีองค์ประกอบหลักดังนี้:

### 1. The Core (แกนกลาง)
* **Aether Conductor:** เปรียบเสมือนระบบประสาทกลาง (Nervous System) คอยรับส่งสัญญาณ (Envelope) ระหว่าง Agent ต่างๆ
* **Akashic Record:** ระบบบันทึกความจริงที่ไม่สามารถแก้ไขได้ (Immutable Ledger)

### 2. The Agents (ประชากร)
* **🧠 Agio Sage (Wisdom):** Agent ผู้ใช้ปัญญา วิเคราะห์ความเสี่ยงและตัดสินใจ
* **🛡️ GEP Enforcer (Immune System):** ผู้คุมกฎ ตรวจสอบคำสั่งตาม `inspirafirma_ruleset.json` (เช่น การโอนเงินเกินวงเงิน)
* **⚙️ Resource/Analysis Agents:** ผู้ปฏิบัติงานจริงในการขับเคลื่อนระบบ

### 3. The Rituals (พิธีกรรม)
* **Commit Ritual:** กระบวนการบันทึก Code ที่ต้องระบุ "เจตจำนง" (Intent) และประทับตราด้วย Cryptographic Signature เพื่อความโปร่งใส

---

### 4. Aetherium Core (Microkernel Preview)
* เพิ่มโครงสร้าง `aetherium_core/` สำหรับ Core Engine แบบ Modular/Microkernel
* รองรับ Plugin lifecycle + dynamic loading + async event bus
